前進LA! 我的夢幻城市旅遊行

By 吉米 2022-08-13
緣起

今年暑假很開心能夠透過教育部的學海築夢計畫與SparkAmplify合作，藉由這次的美國行來提升自己在未來職場上的重要技能，並增進自己的國際觀，從中累積數位行銷的技能，同時也透過實習的機會來認識自我、探索自我，找尋職涯的方向，從中了解到職場實作與學校理論的差異，以妥善規劃未來的發展。

實習內容介紹
  1. 開發學生遠端學習與實習Edtech，進行個案分析研究了解目前的市場需求並進行平台的規劃。
  2. 針對Edtech平台之教育訓練課程進行設計並與台灣分部共同合作，針對細節進行修改調整。
  3. 設計規劃Edtech平台之行銷影片，讓外界更了解並獲取公司相關資訊。
  4. 協助籌備2022全美總會第28屆年會暨第8屆中文傳承教育國際研討會之書展，讓一般民眾更了解平台的運作模式。
  5. 參訪美國加州當地公司，了解其企業文化，學習如何與客戶溝通並了解他們的需求。
生活體驗

一開始到美國真的不太熟悉，不論是時差或是氣候都給我帶來很大的影響，因此在一開始來的時候很常三更半夜就醒來，也常常覺得皮膚乾燥破皮，明明是夏天卻還要一直擦乳液，不過隨著待的時間越來越久，也漸漸熟悉這邊的各種變化。此外，因為美國的土地較大，不像台灣一樣有很多機車在道路上馳騁，也很難有機會可以租借到便利的交通工具，不過幸好Cathy AKA 翁楊乾媽同一時間也在美國，她很熱心地開車載我們去買生活必需品以及參觀加州當地的風景，我們也自己學習如何搭乘當地的公車以及地鐵，並實際搭乘其到洛杉磯的市中心走走逛逛，甚至還自己搭到美國職棒的道奇球場看球，一圓兒時夢想。

具體效益
  1. 增強自己對於英文的敏感度，慢慢能夠聽懂美國當地人的口音。
  2. 能夠有機會練習自己的英文會話能力，了解自己的不足之處。
  3. 有機會能夠接觸到數位行銷領域，將自己的專長所學與自己感興趣的領域相互結合。
  4. 有機會接觸到有別於台灣的異國文化，提升自己的文化涵養。

心得分享

很開心這次可以參與學海築夢的計畫，來到自己心目中的夢幻城市洛杉磯，更開心的是能夠接觸到自己平常可能沒有機會接觸到的數位行銷領域。因為平常在台灣要找跟數位行銷產業相關的實習或工讀機會都很渺茫，由於自己並不是本科系出身，縱使有過相關的經驗或是修過類似的課程，但大部分的公司仍不會給予太多的機會給有志想要往這領域發展的學生，所以我自己是很珍惜這次的機會。加上還有機會可以來到美國實習，以此擴展自己的國際觀，透過到當地的公司進行參訪以及到美國的各個名勝古蹟採點，來觀察並分析美國為何總是能夠將他們的產品透過行銷包裝地如此精緻且吸引人，並將我的所見所聞帶到這次的實習工作中。這次也非常謝謝SparkAmplify給我這個來到美國實習的機會，在實習的過程中不斷地給我建議，一步一步帶我認識數位行銷的工作內容，引導我該如何做出吸引人且創新的東西，也謝謝絲茜老師在美國給予我們的各種照顧，從找租屋處到買東西都沒忘記過我們，也常常兼職司機載我們去協助籌備書展、去企業參訪，這一個月以來減輕了我們不少負擔，多了份安心，少了份憂心。然而近年來因為疫情以及戰爭的影響，造成美國當地的消費比起以往都還要來得高，因此在經費有限的情況下我們很遺憾不能繼續留在當地，同時也建議未來計畫能夠斟酌每個國家的消費進行經費上面的調整，不然這次光租房子就把經費花得差不多了，更別提交通費跟伙食費仍要自行處理，這對於學生來說可能負擔就會較多一些，不過總而言之還是很開心自己能夠透過學海築夢的計畫實現自己的夢想，希望未來能夠藉由這次美好且收益良多的經驗，幫助自己能夠將這次所學運用在工作職場上，以提升自我價值。
